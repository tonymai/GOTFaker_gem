# GOTFaker

A GOT Faker gem that provides random data from Game of Thrones.

Here are the available methods:


##GOTFaker::Character

* #full_name
* #first_name
* #last_name

##GOTFaker::House

* #name
* #description

##GOTFaker::Geography

* #continent
* #region
* #island
* #sea
* #river
* #ruin
* #road
* #forest
* #mountain

#GOTFaker::Quote

* #tyrion_lannister
* #bad_ass

#GOTFaker::Religion

* #name
* #description
